# Survey Outline Templates

## comparison-figures 大纲

```markdown
## 图 1：对比总览
- Role: overview
- Content: 所有论文核心路径并列展示
- Layout: 水平并列或网格布局，每篇固定颜色
- Key annotations: 每篇一句话方法标签

## 图 2：关键维度对比 — 架构
- Role: dimension-comparison
- Dimension: 架构设计对比
- Layout: 按同一流程图的各步骤对比各论文的做法

## 图 3：关键维度对比 — 实验
- Role: dimension-comparison
- Dimension: 实验表现对比
- Layout: 柱状图 / 雷达图 / 表格形式

## 图 4：演进时间线
- Role: timeline
- Content: 按时间展示各论文的演进关系
```

## survey-deck 大纲

```markdown
## 01. 封面
- Role: cover
- Target: 一句话主题

## 02. 问题定义
- Role: context
- Content: 所有论文共同解决的问题

## 03. 论文概述
- Role: overview
- Content: 每篇论文一句话摘要 + 时间线

## 04. 方法对比 — 整体框架
- Role: comparison-overview
- Layout: 所有方法并列流程图

## 05-N. 方法对比 — 各维度
- Role: mechanism-detail
- Content: 每个对比维度展开 1-2 页

## N+1. 实验对比
- Role: evidence
- Content: benchmark 对比表 + 解读

## N+2. Trade-off 分析
- Role: comparison
- Content: 优势/局限对比表

## N+3. 总结与推荐
- Role: takeaway
- Content: 各场景推荐
```

## survey-article 大纲

```markdown
# 标题

## 1. 引言
- 问题背景
- 为什么需要对比这些工作
- 本文的对比框架

## 2. 各论文概述
- 2.1 Paper A
- 2.2 Paper B
- ...

## 3. 维度对比
- 3.1 问题定义对比
- 3.2 方法路径对比
- 3.3 关键设计对比
- 3.4 实验表现对比

## 4. 综合讨论
- 4.1 各方法适用场景
- 4.2 核心 Trade-off
- 4.3 未来方向

## 5. 结论
```
