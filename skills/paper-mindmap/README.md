# paper-mindmap

**把论文变成可视化的思维导图——快速建立理解骨架。**

## 快速开始

```bash
/paper-mindmap https://arxiv.org/abs/1706.03762
/paper-mindmap /path/to/paper.pdf --mode aigc-mindmap --language Chinese
/paper-mindmap notes.md --mode mermaid-code
```

## 两种模式

| 模式 | 输出 | 适合 |
|------|------|------|
| **mermaid-code** | Mermaid mindmap 代码 | README、Notion、Obsidian、GitHub |
| **aigc-mindmap** | AIGC 思维导图图片 (PNG) | 分享、社交媒体、演示 |

## 标准 5 分支

```
📌 Problem       → 研究问题、动机、局限
⚙️ Method        → 核心方法、创新点、模块
📊 Results       → 关键实验数据、Benchmark
💡 Contributions → 核心贡献、理论意义
🔗 Connections   → 相关工作、代码、资源
```
