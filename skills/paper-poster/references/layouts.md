# Poster Layouts

## conference-wide 布局详解

```
┌────────────────────────────────────────────────────────────┐
│  TITLE BANNER (12%)                                         │
│  Paper Title · Authors · Affiliation                        │
├─────────────────────────────┬──────────────────────────────┤
│  PROBLEM & MOTIVATION (18%) │  METHOD OVERVIEW (40%)        │
│  • Background context       │  ┌─────────────────────────┐ │
│  • Research gap             │  │ 核心方法流程图           │ │
│  • This work's goal         │  │ 输入→处理→输出           │ │
│                             │  │ (最大视觉元素)           │ │
│  METHOD PIPELINE (15%)      │  └─────────────────────────┘ │
│  ┌───────────────────────┐  │  Key annotations             │
│  │ 简化流程小图           │  │                              │
│  │ Step1→Step2→Step3     │  │                              │
│  └───────────────────────┘  │                              │
├─────────────────────────────┴──────────────────────────────┤
│  KEY RESULTS (25%)                                          │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐                   │
│  │ Metric A │ │ Metric B │ │ Metric C │                   │
│  │  XX.X%   │ │  YY.Y%   │ │  ZZ.Z%   │                   │
│  │  +X% imp │ │  +Y% imp │ │  +Z% imp │                   │
│  └──────────┘ └──────────┘ └──────────┘                   │
├────────────────────────────────────────────────────────────┤
│  CONCLUSION + REFERENCES (10%)                             │
│  一句话贡献 · 3 个要点 · 4-6 参考文献                      │
└────────────────────────────────────────────────────────────┘
```

## defense-poster 布局详解

```
┌──────────────────────────────────────┐
│  TITLE BANNER (10%)                   │
│  论文标题 · 作者 · 单位 · 导师       │
├──────────────────────────────────────┤
│  RESEARCH PROBLEM (12%)              │
│  研究背景 → 现有局限 → 本文目标      │
├──────────────────────────────────────┤
│  METHOD OVERVIEW (30%)               │
│  ┌────────────────────────────────┐  │
│  │  方法流程图（居中，最大元素）   │  │
│  └────────────────────────────────┘  │
│  关键步骤标注                        │
├────────────────────┬─────────────────┤
│  EXPERIMENTS (18%) │  RESULTS (18%)  │
│  设置 · 数据集     │  核心数据大号    │
│  Baseline · 指标   │  对比表格        │
├────────────────────┴─────────────────┤
│  CONCLUSION (8%)                      │
│  3 个核心贡献 · 1 句总结             │
├──────────────────────────────────────┤
│  REFERENCES · CONTACT (4%)           │
└──────────────────────────────────────┘
```

## research-showcase 布局详解

```
┌──────────────────────────────────────────────┐
│  HERO TITLE (15%) — 粗体大标题               │
│  一行核心主张                                 │
├──────────────────────┬───────────────────────┤
│  KEY INSIGHT (25%)   │  HERO VISUAL (45%)    │
│  核心创新一句话       │  主视觉图              │
│                      │  (方法/产品/效果图)    │
│  IMPACT NUMBERS      │  最上镜的一个视觉      │
│  Metric X: XX% ↗    │                       │
│  Metric Y: YY% ↗    │                       │
├──────────────────────┴───────────────────────┤
│  APPLICATION + CONTACT (15%)                 │
│  应用场景 · 二维码 · 联系方式                │
└──────────────────────────────────────────────┘
```

## Text Size Reference

| Element | conference-wide | defense-poster | research-showcase |
|---------|----------------|----------------|-------------------|
| Title | 72-96 pt | 60-72 pt | 96-120 pt |
| Section headers | 36-48 pt | 32-40 pt | 48-60 pt |
| Body text | 24-28 pt | 22-26 pt | 20-24 pt |
| Figure labels | 20-24 pt | 18-22 pt | 18-20 pt |
| References | 18-20 pt | 16-18 pt | 16-18 pt |
