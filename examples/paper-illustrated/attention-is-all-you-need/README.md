# Attention Is All You Need — Illustrated

Paper: [Attention Is All You Need](https://arxiv.org/abs/1706.03762)

This example shows the new paper method illustration flow:

1. Read the paper.
2. Recommend the figure plan before generating images.
3. Confirm language, style, and page count.
4. Generate only the figures the user actually wants.

## Recommended Plan

For this paper, the skill recommends 2-4 figures:

| Figure | Style | Purpose |
|--------|-------|---------|
| Transformer overview | `paper-figure` | A polished method figure for README, slides, or article headers |
| Self-attention mechanism | `sketchnote` | A warm explanatory page that makes Q/K/V attention intuitive |
| Multi-head attention | `sketchnote` or `paper-figure` | Optional detail figure for parallel relation learning |
| Encoder-decoder bridge | `paper-figure` | Optional detail figure for sequence-to-sequence decoding |
