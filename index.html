🎨 paper-craft-skills

10 AI-powered skills for academic papers — zero config, one command

★ No API keys needed

⚡ Install in 10 seconds

npx skills add zsyggg/paper-craft-skills

Then ask your AI agent: /paper-comic https://arxiv.org/abs/1706.03762

🧰 All Skills

🎨 paper-comic

Paper → AIGC Method Figures (sketchnote / paper-figure)

figureaigcvisual
/paper-comic paper.pdf --style sketchnote
📄 PNG · 2 styles · interactive workflow

📄 paper-analyzer

Paper → Deep HTML Article (storytelling / academic / concise)

articlehtmlkatex
/paper-analyzer paper.pdf --style academic
📄 HTML · KaTeX · Mermaid · dark mode 🌙

🖼️ paper-deck

Paper → AIGC Visual Slide Deck (PPTX/PDF, 4 styles)

slidesaigcpptx
/paper-deck paper.pdf --style journal-minimal --slides 12
📦 PPTX · PDF · 4 presets · quality gate ✓

🔬 paper-survey

Multi-Paper → Comparison Figures/Slides/Article

surveycomparisonaigc
/paper-survey p1.pdf p2.pdf --format survey-article
📊 3 modes · 5-dimension matrix · color-coded

📋 paper-poster

Paper → 300dpi AIGC Conference Poster

posteraigcprint
/paper-poster paper.pdf --style conference-wide
📐 48×36in · 300dpi · 3 presets · print-ready PDF

📊 paper-summary

Paper → One-Page Visual Summary (infographic/card/timeline)

summaryinfographicaigc
/paper-summary paper.pdf --mode infographic
🖼️ 3 modes · 2:3 / 1:1 / 16:9 · max 200 chars

🧠 paper-mindmap

Paper → Mind Map (Mermaid code / AIGC image)

mindmapmermaidstructure
/paper-mindmap paper.pdf
🌳 5 branches · 2 modes · max 4 levels deep

📝 paper-review

Automated Peer Review — 3 reports + editorial decision

reviewpeer-reviewscoring
/paper-review paper.pdf
📋 5 dimensions · 1-10 scoring · meta review

🌐 paper-slides-html

HTML/CSS Slides — zero-dependency, browser-based presentations

htmlslidesno-aigc
/paper-slides-html paper.pdf --theme academic
📄 3 themes · ~50KB · fully editable · print to PDF

⚡ paper-tldr

Extreme Summarization — one-liner / TL;DR / 3 bullets

tldrsummaryfast
/paper-tldr paper.pdf --level all
📝 3 levels · ≤30/150/20 chars · 10s to read
