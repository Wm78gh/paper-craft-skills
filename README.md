## Why It Feels Different

| Skill | Output | Best for |
|-------|--------|----------|
| **paper-comic** | Method overview figures, mechanism diagrams, visual paper notes | README hero images, videos, explainers, technical posts |
| **paper-analyzer** | In-depth Markdown/HTML articles with multiple writing styles | Deep research breakdowns, blog posts, study notes |

`paper-comic` is no longer a fixed 10-page comic generator. It is now a paper method illustration skill: it reads the paper, proposes the right figures, asks whether you want Chinese or English labels, confirms style and page count, and then draws only what you need.

## Quick Start

### 1. Install

```bash
npx skills add zsyggg/paper-craft-skills
```

### 2. Configure

| Skill | Required |
|-------|----------|
| paper-analyzer | `MINERU_TOKEN` from [mineru.net](https://mineru.net) |
| paper-comic | An available image-generation backend. In Codex, it uses the built-in `imagegen` skill. |

```bash
# For paper-analyzer
pip install requests markdown
export MINERU_TOKEN="your_token"
```

### 3. Use

```bash
# Generate method figures
/paper-comic /path/to/paper.pdf
/paper-comic https://arxiv.org/abs/1706.03762 --style paper-figure --language English --pages 1

# Analyze paper, generate article
Please analyze this paper: /path/to/paper.pdf
```

## paper-comic: Paper Method Illustration

Before generating images, the skill recommends a figure plan:

```text
I recommend 4 figures:
1. Method overview
2. Core mechanism A
3. Core mechanism B
4. Key result

Please confirm:
- Language: Chinese / English / bilingual
- Style: sketchnote / paper-figure
- Scope: overview only, or all 4 figures?
```

### Visual Styles

| Style | Look | Use it when |
|-------|------|-------------|
| **sketchnote** | Warm research notebook, hand-drawn arrows, helpful visual anchors | You want people to understand the paper quickly |
| **paper-figure** | Clean conference-paper method figure, polished modules and arrows | You want a professional README, article, or slide hero |
